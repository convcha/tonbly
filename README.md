# Tonbly

## Development

### Prerequisites

- [Docker](https://docs.docker.com/install/)
- [Docker Compose](https://docs.docker.com/compose/install/)
- [Node.js](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/lang/)

### Setup

```shell
$ git clone https://github.com/devilune/tonbly.git
$ cd tonbly
$ yarn
$ cd db
$ yarn up
```

### Run

```shell
# at root directory
$ yarn dev
```
